Q:1 Discuss the scope of var, let, and const

VAR

  • 1. Variables declared with var are in the function scope.
  • 2. In var hosting is allowed. Hoisting means that you can
    define a variable before its declaration.
  • 3. Reassign value is allowed in var
  • 4. Redeclaration of the variable is allowed in var

LET

  • 1. Variables declared as let are in the block scope.
  • 2. In let hosting is not allowed
  • 3. Reassign value is allowed in let
  • 4. Redeclaration of the variable is not allowed in let

CONST

  • 1. Variables declared as const are in the block scope.
  • 2. In const hosting is not allowed
  • 3. Reassign value is not allowed in cons
  • 4. Redeclaration of the variable is not allowed in const

Q:2 Tell us the use cases of null and undefined

NULL

  • 1. Null is often used to explicitly indicate the absence of a value or to represent an intentionally empty or undefined state for a variable or object property.
  • 2. In databases, null is used to represent missing or unknown data in a specific field of a record.
  • 3. In some cases, null can be returned or used to indicate failure or the lack of a valid result from a function or operation, allowing for error handling.

undefined

  • 1. Variables that have been declared but have not been assigned a value automatically have an undefined value until assigned.
  • 2. When a function is called with fewer arguments than declared parameters, the remaining parameters are automatically set to undefined.
  • 3. If a function doesn't explicitly return a value, it will return undefined.

Q:3 What do you mean by REST API?

A REST API, short for Representational State Transfer Application Programming Interface, refers to a structured framework of guidelines and standards used to design and engage with web services. This framework is a prevalent method for establishing and controlling communications among diverse software applications across the internet. REST APIs are extensively employed to facilitate communication between client applications, including web browsers, mobile apps, or other servers, and resources hosted on servers. This interaction enables the retrieval, manipulation, creation, or deletion of data and functionalities stored on the server.
